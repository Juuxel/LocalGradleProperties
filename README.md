# gradle-local-properties
Used to load local Gradle properties from a `private.properties` file.

## Usage
Groovy:  
```groovy
apply from: "https://raw.githubusercontent.com/Juuxel/gradle-local-properties/master/loader.gradle.kts"
```

Kotlin:  
```kotlin
apply(from = "https://raw.githubusercontent.com/Juuxel/gradle-local-properties/master/loader.gradle.kts")
```
